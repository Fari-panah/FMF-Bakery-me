# FMF Bakery 🍰

FMF Bakery is a student project for an online bakery website.
The project currently focuses on building a modern frontend using React, while the backend is still in progress.

---

## 🚀 Project Status

* ✅ Frontend: Completed (React)
* 🚧 Backend: In progress
* 🗄️ Database: Planned (MongoDB)

---

## 🛠️ Tech Stack

**Frontend**

* React
* JavaScript
* HTML / CSS

**Backend (In Progress)**

* Node.js
* Express.js

**Database (Planned)**

* MongoDB

---
